# 2d-renderer-samples
This repository contains samples for the 2D Renderer in Universal Render Pipeline to be used with Unity 2021.2 or newer.

### Contents:

**Feature samples**

- 2D Light Types
- 2D Shadows
- Normal Maps (Secondary Texture)
- Masks (Secondary Texture)
- Pixel Perfect Camera
- Post Processing
- Shader Graph
- Visual Effect Graph
- Renderer Features

**Applied samples**
- Emissive Shader
- Heat Haze Overlay
